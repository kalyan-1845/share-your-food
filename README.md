# Seva - The Food Sharing App

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![React](https://img.shields.io/badge/react-%5E17.0.2-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

An innovative, user-friendly application designed to bridge the gap between food donors and NGOs. **Seva** connects individuals wanting to donate surplus food with non-governmental organizations that serve those in need.

Developed by **Bhoompally Kalyan Reddy**.

---

## 🌟 Key Features

- **NGO Selection**: Easily select from a curated list of verified NGOs to donate food.
- **Portion Specification**: Specify the exact number of people the food can serve.
- **Dietary Choices**: Clearly classify donations as Vegetarian or Non-Vegetarian.
- **Condition Marking**: Choose to donate either freshly cooked meals or raw groceries.
- **Flexible Delivery**: Opt for either self-delivery to the NGO or request a convenient pickup.
- **Secure Authentication**: Robust user authentication to keep data secure.
- **Mobile-Responsive**: Fluid UI/UX designed to work flawlessly across desktop, tablet, and mobile devices.

## 🛠️ Technology Stack

- **Frontend**: ReactJS, CSS, Styled-JSX
- **Backend Infrastructure**: ExpressJS, MongoDB
- **Deployment**: Vercel ready

## 🚀 Getting Started

Follow these instructions to get a copy of the project running on your local machine for development and testing purposes.

### Prerequisites

- Node.js (v14 or above is recommended)
- npm or yarn

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kalyan-1845/seva-the-food-sharing-app-main.git
   ```

2. **Navigate to the frontend directory:**
   ```bash
   cd seva-the-food-sharing-app-main
   ```

3. **Install Dependencies:**
   Using npm:
   ```bash
   npm install
   ```
   Or using yarn:
   ```bash
   yarn install
   ```

4. **Initialize the App (Development Mode):**
   ```bash
   npm start
   ```

The application will spin up at `http://localhost:3001` or `http://localhost:3000`. The project will automatically reload when source files are modified.
